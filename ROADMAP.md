# Roadmap - @phdiniiz/comercialTime

Este documento descreve o roadmap e planos futuros para a biblioteca @phdiniiz/comercialTime.

## 🎯 Visão Geral

Nosso objetivo é tornar @phdiniiz/comercialTime a biblioteca de referência para gerenciamento de horários comerciais em português brasileiro, expandindo para suporte global e funcionalidades avançadas.

## 📅 Cronograma

**Foco: Expansão Geográfica**
